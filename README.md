# Covenant of Shadows

A dark gothic tale of love, friendship and betrayal.

## Overview

Welcome to **Covenant of Shadows** — an atmospheric, dark gothic narrative experience. This project weaves together themes of love, friendship, and the tragic consequences of betrayal in a haunting digital tale.

## Project Structure

- **HTML (80.8%)** - Core narrative and interactive content
- **CSS (19.2%)** - Styling and visual atmosphere

## Features

- Dark gothic aesthetic and storytelling
- Interactive narrative elements
- Immersive visual design with custom styling

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Khmerix/-Covenant-of-Shadows.git
   ```

2. Open the project in your preferred browser or text editor

3. Explore the story and experience the tale unfold

## Development

This project is built with:
- **HTML5** for semantic structure and content
- **CSS3** for atmospheric styling and design
